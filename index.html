¿Qué es el algoritmo de Huffman?

La codificación de Huffman es una técnica para la compresión de datos mediante el estudio de la frecuencia de aparición de caracteres, la cual es ampliamente usada y muy efectiva. Fue desarrollado por el norteamericano David Albert Huffman en 1952 mientras hacía el doctorado en el MIT. El método fue publicado en una revista como “A Method for the Construction of Minimum-Redundancy Codes” el cual puedes consultar aquí.

¿Cómo funciona el algoritmo de Huffman?

Huffman inventó un algoritmo voraz que construye un árbol de codificación optima, del cual, se obtiene la codificación optima de un mensaje dado. El algoritmo funciona a partir de un alfabeto de n símbolos, junto con sus frecuencias de aparición asociadas y produce un código de Huffman para esas frecuencias y alfabeto.

El algoritmo consiste en la creación de un árbol binario que tiene cada uno de los símbolos por hoja, y construido de tal forma que, siguiéndolo desde la raíz a cada una de sus hojas se obtiene el código Huffman asociado.

Pasos del algoritmo

Para obtener el árbol de codificación optimo:

Para cada carácter único del alfabeto, crear un nodo hoja y crear una cola donde el carácter menos frecuente es el primero (cola de prioridad).

Extraer dos nodos con la frecuencia mínima de la cola de prioridad.

Cree un nuevo nodo con la frecuencia igual a la suma de las frecuencias de los dos nodos. El primer nodo extraído será su hijo izquierdo y el otro nodo extraído como hijo derecho. Ingresar el nodo a la cola de prioridad nuevamente.

Repita los pasos 2 y 3 hasta que la cola de prioridad contenga solo un nodo. El nodo restante es el nodo raíz y el árbol está completo.

Para obtener los códigos óptimos por cada carácter del alfabeto:

Iniciar siempre el recorrido en el árbol formado a partir de la raíz. Tener una matriz auxiliar.

Mientras nos movemos a la izquierda escribir 0.

Mientras nos movemos a la derecha escribir 1.

Si llegamos a una hoja imprima la matriz.

Complejidad

Este algoritmo tiene una complejidad de O(nlog(n))

¿Por qué es un algoritmo voraz (Greedy)?

El algoritmo de codificación de Huffman cumple con los siguientes elementos, los cuales caracterizan a una técnica voraz:
Conjunto de candidatos
Cada uno de los nodos dentro de la cola de prioridad
Función solución
Unión de nodos en un nuevo árbol
Función de selección
Selección de nodos con menor frecuencia
Función de factibilidad
La cola de prioridad debe de tener mas de un elemento
Función objetivo
Árbol de codificación óptimo

¿Para que sirve este algoritmo?

El algoritmo de codificación de Huffman se usa ampliamente en todos los formatos de compresión convencionales que puede encontrarse, desde GZIP, PKZIP (winzip, etc.) y BZIP2, hasta formatos de imagen como JPEG y PNG.
